Reporting SQL
Base de données films.

Interrogation SQL et visualisation graphique sous Excel

A

Objectifs du programme national

Dans le cadre du programme national du BUT Science des Données, la SAÉ 101 a pour vocation de faire découvrir les bases de données relationnelles et les fondements du langage SQL. Elle s'inscrit dans la compétence Traiter — gérer le cycle de vie de la donnée — au niveau 1 (BUT1).

Les objectifs visés par le PN pour cette SAÉ sont :

  • Comprendre la structure d'une base de données relationnelle (tables, clés, schéma)
  • Maîtriser les requêtes SQL fondamentales : SELECT, JOIN, GROUP BY, ORDER BY
  • Produire un reporting lisible à partir de données brutes
  • Communiquer des résultats via des visualisations graphiques adaptées
B

Problématique & livrables

La problématique imposée était la suivante : à partir d'une base de données films fournie sous Microsoft Access (tables Films, Acteurs, Réalisateurs, Notes), comment extraire des informations métier pertinentes via SQL et les restituer sous forme de rapport graphique ?

Il s'agissait de démontrer sa capacité à naviguer dans un schéma relationnel inconnu, à formuler des questions analytiques, puis à y répondre par des requêtes appropriées.

Livrables :
Requêtes SQL documentées Tableau de bord Excel Rapport de synthèse
Logiciels :
SQL Microsoft Access Excel
C

Méthodologie

01
Exploration du schéma

Identification des tables, clés primaires et étrangères, compréhension des relations entre entités (Films ↔ Acteurs ↔ Notes).

02
Rédaction des requêtes SQL

Progression des requêtes simples (sélections, filtres) vers les requêtes complexes (jointures multiples, agrégations COUNT/AVG/MAX).

03
Export et mise en forme

Export des résultats vers Excel, structuration des données pour la visualisation.

04
Reporting graphique

Création de graphiques commentés : genres les mieux notés, top acteurs, évolution des sorties par année.

D

Résultats obtenus

Le projet a abouti à un ensemble de requêtes SQL documentées couvrant différents niveaux de complexité, ainsi qu'un tableau de bord Excel comportant plusieurs visualisations commentées.

  • Classement des genres par note moyenne (histogramme)
  • Top 10 des acteurs les plus présents dans la base
  • Évolution du nombre de sorties de films par année
  • Rapport de synthèse présentant la démarche et les conclusions
