Prix des logements
Boston Housing.

Régression linéaire multiple sur 506 observations — modèle à 7 variables sélectionné par 4 méthodes convergentes, RMSE = 3,63 k$

A

Objectifs du programme national

La SAÉ 401 mobilise les compétences Analyser, Modéliser et Valoriser au niveau 2 autour de la prédiction d'une variable quantitative continue par régression linéaire multiple. Elle s'inscrit dans le parcours EMS (Exploration et Modélisation Statistique) du BUT SD.

Les objectifs visés par le PN pour cette SAÉ sont :

  • Conduire une analyse exploratoire complète (univariée, bivariée) pour orienter la modélisation
  • Construire et comparer plusieurs modèles de régression par montée en complexité
  • Sélectionner les variables par méthodes pas à pas (Forward, Backward, Stepwise) et exhaustive
  • Évaluer la qualité prédictive par validation croisée K-fold et critère RMSE
  • Vérifier les hypothèses du modèle linéaire (MCO) et interpréter les violations dans leur contexte
B

Problématique & livrables

À partir du dataset Boston Housing (506 observations de zones urbaines, 12 variables socio-économiques, environnementales et structurelles), peut-on construire un modèle linéaire multiple performant pour prédire le prix médian des logements (medv, en k$) — avec le RMSE en validation croisée comme unique critère de sélection ?

Contrainte identifiée dès l'analyse : les observations à medv = 50 k$ sont censurées (plafond réel du marché) → retrait → 490 observations retenues. Multicolinéarité forte tax/rad (r = 0,91) → tax retirée (VIF = 8,82 > rad VIF = 7,33).

Livrables :
Rapport de modélisation (19 p.) 7 modèles comparés Prédictions par profil (IC 95 %)
Logiciels :
R / RStudio GGally (ggpairs) leaps (regsubsets) car (VIF) lmtest · caret
C

Méthodologie

01
Analyse exploratoire — nettoyage et détection des structures

Analyse univariée de medv : distribution non normale (Shapiro-Wilk), seuil de censure à 50 k$ → retrait des observations tronquées → 490 obs. retenues. Matrice ggpairs() sur les 12 variables : corrélation rad/tax = 0,91 (multicolinéarité sévère → tax supprimée via VIF), non-linéarité visible entre lstat et medv (courbure dans le nuage de points) → introduction du terme lstat².

02
Modèles de découverte — montée en complexité

M1 : medv ~ rm — modèle simple linéaire (R² = 47 %, RMSE = 5,71 k$) : +8 270 $/pièce. M2 : medv ~ lstat + lstat² — capture la non-linéarité (R² = 66 %, RMSE = 4,58 k$) : -1 955 $/1 % de pop. défavorisée, effet qui s'atténue. M3 : modèle complet sans tax (11 variables + lstat²) : RMSE = 3,62 k$ — meilleur absolu mais 12 prédicteurs.

03
Sélection de variables — 4 méthodes, critère RMSE en validation croisée K-fold

Forward (sélection progressive), Backward (élimination progressive), Stepwise (ajout/retrait combiné), Exhaustive (regsubsets() — toutes combinaisons). Point d'inflexion identifié à 7 variables : au-delà, le RMSE se stabilise (overfitting potentiel). Les 4 méthodes convergent vers les mêmes 7 variables : crim, nox, rm, dis, ptratio, lstat, lstat². RMSE moyen en K-fold : ~3,63 k$.

04
Diagnostics et prédictions du modèle final

Vérification des 5 hypothèses MCO : linéarité (OK par construction), homoscédasticité (Breusch-Pagan rejeté — fréquent sur données immobilières, impact limité), indépendance (Durbin-Watson : légère autocorrélation spatiale), normalité des résidus (Shapiro rejeté, TCL valide pour n=490), multicolinéarité (VIF OK sauf couple structurel lstat/lstat²). Prédictions par profil type avec intervalles à 95 %.

D

Résultats obtenus

Le modèle final medv ~ crim + nox + rm + dis + ptratio + lstat + lstat² atteint un RMSE de 3,63 k$ en validation croisée — soit une erreur de prédiction moyenne de 3 630 $ sur le prix médian d'un logement bostonien.

  • lstat (% pop. défavorisée) : coefficient standardisé ≈ −9 — prédicteur dominant négatif, relation non linéaire avec lstat²
  • rm (nb moyen de pièces) : coeff. std ≈ +5 — seul prédicteur positif dominant
  • nox (pollution) : coeff. std ≈ −4 · ptratio (ratio scolaire) : coeff. std ≈ −3 · dis (distance emploi) : coeff. std ≈ −2
  • Prédictions : quartier aisé 33,3 k$ [26,1 ; 40,5] · moyen 22,5 k$ [15,3 ; 29,6] · défavorisé 14,3 k$ [7 ; 21,4]
  • Écart aisé/défavorisé : ~25 k$ — entièrement expliqué par les variables socio-économiques et environnementales
